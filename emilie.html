About this Timeline

This site was created by Alexander Kohlhofer. Feel free to check out the source and setup.

Retrieving content from a public Google Document the application automatically generates an interactive timeline. This happens in the client without any server side code and relies entirely on Timeline to do the heavy lifting. It uses some Bootstrap magic, too. The latest versions of these packages are integrated as submodules.

This site was build in a single day and is totally free (free software, free hosting, free everything). You should make one, too.
